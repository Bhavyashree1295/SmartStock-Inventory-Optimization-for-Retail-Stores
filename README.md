SmartStock is a simple Streamlit-based dashboard designed to help retail and grocery stores monitor inventory, analyze sales trends, and identify products that may need restocking. Using the Grocery Inventory and Sales Dataset, the app automatically detects key columns, visualizes sales over time, highlights top-selling items, compares inventory vs sales, and provides quick reorder suggestions based on recent demand. It offers a clean, beginner-friendly interface that turns raw inventory data into actionable insights for smarter stock management.
